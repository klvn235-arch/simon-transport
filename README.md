# Simon Transport

## Project Description
Simon Transport is an innovative logistics platform designed to streamline the shipment of goods across various regions. The service is tailored to meet the needs of both small businesses and large enterprises, offering a user-friendly interface and efficient tracking capabilities.

## Features
- **Real-time Tracking:** Users can monitor their shipments in real time, providing peace of mind and transparency.
- **Dynamic Routing:** Our algorithms find the most efficient routes, reducing shipping time and costs.
- **User-Friendly Dashboard:** Manage shipments, view past orders, and access reports all in one place.
- **Flexible Payment Options:** Multiple payment options are available to accommodate our clients' needs.
- **Customer Support:** Dedicated support team available 24/7 to assist with any inquiries.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/klvn235-arch/simon-transport.git
   ```
2. Navigate to the project directory:
   ```bash
   cd simon-transport
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
5. Visit [http://localhost:3000](http://localhost:3000) to view the application.
